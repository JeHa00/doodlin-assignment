- 해당 포지션의 과제 전형은 주어진 요구 사항을 읽고 이를 구현하여 제출하는 과제입니다.

  - 추가로 시간을 요청할 경우 2일, 48시간까지 추가로 지급이 가능합니다. (우선 기본 제공시간에 진행하되 추가시간이 필요하시다면 하루 전에 요청 주시면 됩니다.
